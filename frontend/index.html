กดปุ่มใช้ฟังก์ชันจริงที่อาจารย์สอน — COS3401 / COS4301 Digital Image Processing

ใช้ยังไง

  1. เลือกรูป จากแถบซ้าย (หรือลากไฟล์ของตัวเองมาวาง)
  2. กดฟังก์ชัน ที่อยากลอง แล้วปรับสไลเดอร์ดูผลทันที
  3. ดู histogram ข้างล่างว่าเปลี่ยนไปยังไง ลากเส้นแบ่งกลางภาพเพื่อเทียบก่อน/หลัง
  4. เปิดแท็บ คำนวณทีละขั้น เพื่อดูตัวเลขทุกขั้น และ โค้ด เพื่อดูคำตอบข้อสอบ
  5. กด + ต่อฟังก์ชัน เพื่อทำหลายขั้นต่อกัน เช่น Gray → Median → Otsu
